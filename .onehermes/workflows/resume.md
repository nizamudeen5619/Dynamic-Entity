# Resume Workflow

**Purpose**: To ensure the AI assistant does not start from scratch every session and maintains long-term project context.

**Execution Steps**:
1. Read `n:\Work\backup\.antigravityrules` to refresh custom system constraints.
2. Read `n:\Work\backup\MEMORY.md` to load the previous state, context, and project history.
3. Acknowledge the loaded context and ask the user how to pick up from the last recorded work log.
